# CodeWave

A modern digital agency website built with HTML, CSS, JavaScript, and GSAP.

## About This Project

CodeWave is a custom PHP-based digital agency website developed for a client in Karachi, Pakistan. The project features a modern UI/UX, smooth animations powered by GSAP, responsive design, and performance-focused development to deliver a professional online presence.

### Technologies Used

* Custom PHP
* HTML5
* CSS3
* JavaScript (ES6+)
* GSAP (GreenSock Animation Platform)
